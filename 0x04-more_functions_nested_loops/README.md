# 0x04. More functions, more nested loops

## Description
This project covers writing functions and using nested loops in C. It includes character checks, arithmetic, printing patterns, and the classic FizzBuzz problem.

## Files

| File | Description |
|------|-------------|
| `0-isupper.c` | Checks for uppercase character |
| `1-isdigit.c` | Checks for a digit (0 through 9) |
| `2-mul.c` | Multiplies two integers |
| `3-print_numbers.c` | Prints numbers 0 to 9 |
| `4-print_most_numbers.c` | Prints numbers 0 to 9, except 2 and 4 |
| `5-more_numbers.c` | Prints numbers 0 to 14, ten times |
| `6-print_line.c` | Draws a straight line using `_` |
| `7-print_diagonal.c` | Draws a diagonal line using `\` |
| `8-print_square.c` | Prints a square using `#` |
| `9-fizz_buzz.c` | Prints numbers 1 to 100 with FizzBuzz rules |
| `10-print_triangle.c` | Prints a triangle using `#` |
| `main.h` | Header file with all function prototypes |

## Compilation
All files are compiled on Ubuntu 20.04 LTS using:
```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89
```
